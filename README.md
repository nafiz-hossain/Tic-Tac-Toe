# ReactJS Tic Tac Toe

## Overview

This is a simple implementation of the classic game Tic Tac Toe using ReactJS. This project is perfect for those who are just starting out with React and want to learn more about its fundamental concepts.

## Features

- **2 Player Game:** Challenge your friends and see who comes out on top!
- **Interactive UI:** A clean and intuitive interface that makes playing easy and fun.
- **Move History:** Keep track of each move made during the game.

## Installation

Follow these steps to get the project up and running on your local machine:

1. Clone the repository:
    ```bash
    git clone https://github.com/nafiz-hossain/Tic-Tac-Toe.git
    ```

2. Navigate into the project directory:
    ```bash
    cd Tic-Tac-Toe
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

4. Start the development server:
    ```bash
    npm start
    ```

Now, open your browser and navigate to `http://localhost:3000` to see the application in action!
